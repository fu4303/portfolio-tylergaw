  • Jribbble

    When a popular web service releases a public API it's like the dinner bell is ringing. Web developers everywhere rush to start consuming that open data as quick as possible. Jribbble was my reaction to Dribbble.com releasing their API. It is a jQuery plugin that can be used to easily pull shot and player information that is available from the web service.
