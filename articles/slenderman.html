Slenderman

29 Jun 2013

By Tyler Gaw

Fonts have always felt like a black box to me. The thought of creating a new one has seemed like a task reserved for a select few with decades of training and a deep knowledge of the history of typography. Certainly not me. I've shown myself that that is not entirely true. Slenderman is my first typeface. I made to fit the needs of a project I'm working on, and of course, for fun.

Why Do This?

I made this font to scratch an itch. Specifically, I started a small project that called for hand drawn type. I could have gone the route of using images, but that just doesn't feel right on the Web these days. I want my web type to be scaleable and maintainable. Using images for type is neither of those. It was clear that I'd need to either find an existing font that fit the style I wanted, or roll my own.

I didn't look around for existing fonts that fit. I immediatly jumped to creating my own. The project is a personal one so there's no real time limit. When I have that choice I love to let projects expand to as many different areas as they need to. This same project also led me down the path of building another small tool.
