CSS True Titles

By Tyler Gaw

The opening titles to the show True Detective are incredible. CSS contains most of the building blocks necessary to recreate a similar title sequence. So I tried my hand it and came up with interesting results. Here's the demo and the source code. Read on for details about how it came together.

The first thing I should do is issue the disclaimer. This is an experimental project. Things will most likely go wrong when you view it. Also, due to limited suppport of CSS masking you'll only see the full effect in Chrome and possibly Mobile Safari. The animations will work in other browsers, but things will look strange.

There are a lot of things going on in each shot of the TD titles. The main effect that I wanted to recreate was the use of a person to mask another shot. They use video, but I stuck with still images.

In the demo I'm accomplishing masking using the CSS mask property. As I mentioned above, it has limited support. Here's an example of masking in use. This the CSS to mask the big face image on the photograpy title card. 

.photography .composition {
  -webkit-mask: url("../img/photography-mask.png") top left/cover no-repeat;
  mask: url("../img/photography-mask.png") top left/cover no-repeat;
}

And the mostly transparent image used to mask the content. The transparent portions will not be visible, the opaque portions will be.
