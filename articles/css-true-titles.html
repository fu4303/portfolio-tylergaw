CSS True Titles

By Tyler Gaw

The opening titles to the show True Detective are beautiful. CSS has most of the building blocks necessary to create a similar title sequence. I tried my hand it and came up with interesting results. Here's the demo and the source code.

The first thing I have to do is issue the disclaimer. This is an experimental project. Things will most likely go wrong when you view it. I'm trying to make the browser do things that it really doesn't want to. Also, due to limited suppport of CSS masking you'll only see the full effect in Chrome and possibly Mobile Safari. The animations will work in other browsers, but things will look strange.

There's a lot going on in each shot of the TD titles. The main effect that I wanted to recreate was the use of human silhouettes to mask shots. They use video, but in my demo I stuck with still images.

I'm accomplishing masking using the CSS mask property. As I mentioned above, it has limited support. This the CSS to mask the big face image on the photograpy title card. 

.photography .composition {
  -webkit-mask: url("../img/photography-mask.png") top left/cover no-repeat;
  mask: url("../img/photography-mask.png") top left/cover no-repeat;
}

And the mostly transparent image used to mask the content. The transparent portions will not be visible, the opaque portions will be.
