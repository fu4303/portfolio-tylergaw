CSS True Titles

By Tyler Gaw

The opening titles to the show True Detective are beautiful. CSS has most of the building blocks necessary to create a similar title sequence. I tried my hand it and came up with interesting results. Here's the demo and the source code.

The first thing I have to do is issue the disclaimer. This is an experimental project. Things will most likely go wrong when you view it. I'm trying to make the browser do things that it really doesn't want to. Also, due to limited suppport of CSS masking you'll only see the full effect in Chrome and possibly Mobile Safari. The animations will work in other browsers, but things will look strange.

The Masking

There's a whole lot going on in each shot of the TD titles. You should read the write up about it on Art of the Title. The main effect that I wanted to recreate was the use of human silhouettes to mask shots. They use video clips for the masks and the content being masked, but I stuck with still images to keep things a little easier.

I'm accomplishing masking using the CSS mask property. As I mentioned above, it has limited support. I'm going to break down the first title card to explain how the masking is implemented.

Each title card is an li element. They contain some base styles like width, height, positioning, etc. Within each li is markup specific to that card. Below is the markup for images of the opening card. 

<div class="composition animated">
  <div class="town animated"></div>
  <div class="human"></div>
</div>

The containing div element with the class of "composition" houses all the images. I'm using a containing element here so I can move and scale the "human" and the "town" together, while still being able to move and scale each independently. I'm also applying the mask to the composition element. The "human" and "town" elements have background images applied to them.

I want to be able to move the "town" element. If I applied the mask to that element it would move with it and would not line up properly with the human image. I needed to apply the mask to an element above it. Since CSS mask are only concerned with the opaque and transparent portions of an image, I can reuse the human image as the mask for the "composition" element using the CSS below. 

.opener .composition {
  ...
  -webkit-mask: url("opener-human.png") top left/cover no-repeat;
  mask: url("opener-human.png") top left/cover no-repeat;
}
